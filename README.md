## 🔍 Overview

This repository contains the deployment code for a **real-time gold price prediction system**. The model predicts the next trading day's gold futures price (GC=F) using live market data and online learning.

This is an **independent extension** of a team project. The original model was built collaboratively. I extended it by:
- Adding **online learning** so the model updates itself daily with new market data
- Building and deploying a **live web application** on Streamlit Cloud
- Implementing **MAPE-weighted learning** to adjust how aggressively the model learns based on its current confidence

---

## 🚀 Features

- **Next-day gold price prediction** updated every trading day
- **Online learning** — model continuously improves with new data (no retraining needed)
- **Two-tier update strategy:**
  - Daily: SGD weights updated with new prices
  - Monthly: Scaler recalibrated as gold prices move to new ranges
- **MAPE-weighted `partial_fit`** — model learns more aggressively when less confident
- **Auto-runs once daily** — first visitor triggers the update, everyone else loads from cache
- **Model accuracy metrics** — MAE and MAPE displayed live in the UI

## ⚙️ How It Works

```
On first open each day:
  1. Load sgd_pipeline.pkl (trained in notebook)
  2. Fetch new market data since last training date
  3. Update SGD weights via partial_fit (online learning)
  4. Update scaler every 30+ days (prevents drift)
  5. Predict next trading day price
  6. Save results to gold_data_all.pkl

On subsequent opens same day:
  → Load directly from gold_data_all.pkl (no API calls)
  → All users see same result instantly
```

---

## 🔒 Stability Safeguards

- **sklearn version guard** — detects version mismatch before `partial_fit` runs
- **Sanity check** — predictions outside $1,000–$15,000 trigger automatic cache rebuild
- **Retry with backoff** — Yahoo Finance rate limits handled automatically (up to 5 retries)
- **Monthly scaler recalibration** — prevents long-term drift as gold prices move to new ranges
- **FRED fallback** — if FRED API is unreachable, last known Fed Rate is used
